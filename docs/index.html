What is a derivative?

A derivative measures how fast something is changing. If x(t) is where an object is, then:

v = dx/dt = lim[Δt→0] Δx/Δt

…is how fast the position is changing — the velocity.

And the second derivative?

Apply the same idea to velocity:

a = dv/dt = d²x/dt²

…is how fast the velocity is changing — the acceleration.

What to notice in the charts

→ When the position curve peaks or dips, the velocity crosses zero.

→ When velocity is at its peak, acceleration is zero.

→ The steeper the position slope, the larger the velocity.

How the ESP calculates this

We use a central difference approximation — more accurate than a forward difference because the errors cancel out:

v(t) ≈ [x(t+Δt) − x(t−Δt)] ───────────────────── 2 · Δt
a(t) ≈ [x(t+Δt) − 2x(t) + x(t−Δt)] ───────────────────────────────── Δt²

Error analysis

The central difference has error O(Δt²). With Δt = 50ms, the truncation error is ~2.5ms² order — small but visible as slight noise on the acceleration curve.

Why is acceleration noisier?

Each differentiation amplifies noise. Even a 1mm measurement error becomes ~400 mm/s² error after two differentiations at Δt=50ms. This is why real systems use filters (Kalman, low-pass) after differentiating.
