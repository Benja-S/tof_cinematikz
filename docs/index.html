🧪 ¿Qué es una derivada?

La derivada mide qué tan rápido cambia algo. En este laboratorio, usamos un sensor de tiempo de vuelo para medir la posición, y luego calculamos la velocidad y la aceleración usando cálculo.

x(t)  →  v = dx/dt  →  a = dv/dt

El ESP8266 calcula las derivadas usando el método de diferencia central:

v ≈ [x(t+Δt) − x(t−Δt)] / (2·Δt)
a ≈ [x(t+Δt) − 2x(t) + x(t−Δt)] / Δt²

Δt = intervalo de muestreo (≈50ms)

📊 Cómo leer las gráficas

● Posición — Dónde está el objeto. La pendiente de esta curva es la velocidad.
● Velocidad — Qué tan rápido cambia la posición. Cuando v = 0, el objeto está momentáneamente en reposo.
● Aceleración — Qué tan rápido cambia la velocidad. Cuando a = 0, la velocidad está en su máximo o mínimo.
